 # Attrition-Analysis-Dashboard (Power BI)
Employee Attrition Analysis Dashboard Built in Power BI to identify key factors influencing workforce turnover and Support data -driven HR decision-making.

 # Project Overview 
Employee Attrition Significantly impacts Organizational Productivity,Hiring Costs,and Long-term Growth.
This Project Analyzes attrition patterns across multiple HR dimensions to help Stakeholders Understand :
 -Overall attrition rate and workforce distribution 
 -Attrition by department ,job Role and Education Field 
 -Impact of salary age and Experience on attrition
 -Gender-Wise and dempgrahic attrition tends 
 -key Drivers contributing to employee turnover 
The Dashboard enavbles HR Team and Business Leaders to quickly Identify risk areas and improve retention strategies.
